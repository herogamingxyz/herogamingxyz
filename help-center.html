How To Download Mods

Follow these simple steps to install Minecraft mods safely on your PC.

  • ✔ Download Fabric or Forge Loader
  • ✔ Download your favorite mod
  • ✔ Open Minecraft mods folder
  • ✔ Move mod file into the folder
  • ✔ Launch Minecraft using the correct profile
  • ✔ Enjoy your new mod features

How To Install Shaders

Shaders improve Minecraft graphics and lighting effects. You can use Iris Shaders or OptiFine for shader support.

  • ✔ Install Iris Shaders or OptiFine
  • ✔ Download a shader pack
  • ✔ Open shaderpacks folder
  • ✔ Move shader zip file into folder
  • ✔ Enable shaders from video settings
  • ✔ Restart game if needed

FPS Boost Tips

Low FPS can ruin gameplay experience. These tips can help improve Minecraft performance.

  • ✔ Use Sodium Mod
  • ✔ Lower render distance
  • ✔ Disable fancy graphics
  • ✔ Use lightweight shaders
  • ✔ Close background apps
  • ✔ Allocate proper RAM to Minecraft
